# Escape Room Game

A Python-based Escape Room puzzle game built with Tkinter. The player must solve a series of riddles to escape the room. The game includes images, text-to-speech narration, background music, and automatic save/load functionality.

## Features

* Interactive GUI built with Tkinter
* 5 escape-room riddles
* Image-based room scenes
* Text-to-speech narration using pyttsx3
* Background music
* Progress tracking
* Automatic save and load system using JSON
* Restart game option

## Technologies Used

* Python
* Tkinter
* Pillow (PIL)
* pyttsx3
* JSON
* Winsound

## Installation

1. Clone or download the project.
2. Install the required packages:

pip install -r requirements.txt

## Run the Game

python main.py

## How to Play

1. Launch the game.
2. Type "start" to begin.
3. Solve each riddle by entering the correct answer.
4. Progress through all five riddles.
5. Escape the room and win the game.
